Setting up a NodeJS webserver

Download and install NodeJS

Download and install NodeJS from their website

Initinalize project folder

after the installation is done, open command prompt/command line and navigate to where the project file is located and type: 

                    
        npm init

Setting the base filepath to the project folder

NodeJS has a built in variable to get the filepath that the NodeJS file is running from: __dirname

Shown below is the code in a node.js file. 

                    
    var path = require('path');
    var options = {root: path.join(__dirname, 'docs')};
